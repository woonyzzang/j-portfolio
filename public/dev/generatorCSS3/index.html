  • @font-face
    ?
    Generate CSS3 Font Face
    웹 사이트가 아닌 웹 적합 글꼴을 사용할 수 있습니다.
    파일첨부시 확장자 .OTF 나 .TTF 폰트파일이 필요합니다 그리고 웹 사이트에 글꼴을 포함 할 수있는 권한이 있어야 합니다.
    Browser Support
    IE, Chrome, Safari, Firefox, Opera

    • 글꼴을 변환하여 당신의 글꼴 라이센스를 확인하고 그것이 웹 사이트에 글꼴을 포함 할 수 있습니다. (Font2Web에서 제공)

  • Backface Visibility
    ?
    Generate CSS3 Backface Visibility
    화면에 직면하지 않을 경우 요소가 표시되어야하는지 여부를 정의합니다.
    요소가 회전 할 때와 뒷면을 보고 싶지 않을때 이 속성은 유용합니다.

    IE 10+, Chrome, Safari, Firefox, Opera

  • Opacity
    ?
    Generate CSS3 Opacity
    요소를 투명 반투명 또는 불투명하게 만들 수 있습니다.
    값은 (0-1)을 사용합니다. 0 = 완전 투명, 0.5 = 50 %의 투명, 1 = 불투명.
    IE 하위 브라우저에서는 CSS3의 Opacity 속성값을 지원하지 않아 filter 속성을 사용합니다.
    Browser Support
    IE, Chrome, Safari, Firefox, Opera
  • Perspective
    ?
    Generate CSS3 Perspective
    원근감을 제어합니다. 값이 작을 수록 가까이에서 보는 것 같이 느껴지고 3D 효과도 커집니다.
    부모요소에 지정할 경우 자식요소들이 동일한 소실점을 갖습니다.
    CSS3 3D 속성에 사용됩니다.
    Browser Support
    IE 10+, Chrome, Safari, Firefox, Opera

  • Transition
    ?
    Generate CSS3 Transition
    2가지 스타일 사이의 요소를 전환 할 수 있습니다.
    전환을 사용하는 경우에만 요소를 기본 클래스에 전환을 설정해야합니다. 이 작업을 수행 한 후 모든 :hover 클래스는 애니메이션 켜고 자동으로 됩니다. 또한 추가하고 자바 스크립트와 요소가 스타일 사이 전환됩니다를 사용하여 요소에 새 클래스를 제거 할 수 있습니다.
    Browser Support
    IE 10+, Chrome, Safari, Firefox, Opera
