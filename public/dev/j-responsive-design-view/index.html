How it works

반응형 웹 개발시 각 디바이스별 해상도의 기본 브라우저 실제크기 화면을 확인할수 있는 웹 어플리케이션 테스트 도구 입니다. 디바이스의 운영체제별 특성, 버그는 실제 디바이스를 통해 확인하세요.

기능 요약

ENTER URL:
특정 페이지를 테스트하려면 주소 표시 줄에 웹 사이트의 URL 주소를 입력 할 수 있습니다.
Single View:
각 디바이스 카테고리 메뉴의 해상도를 확인할 수 있습니다.
Custom View:
사용자가 직접 해상도 설정을 정의하여 확인할 수 있습니다.
Horizontal Alignment:
멀티 디바이스 정렬기준을 가로로 정렬하여 확인할 수 있습니다.(멀티 뷰 모드일시 활성화)
Vertical Alignment:
멀티 디바이스 정렬기준을 세로로 정렬하여 확인할 수 있습니다.(멀티 뷰 모드일시 활성화)
Multi View:
모든 디바이스 종류의 해상도를 한눈에 확인할 수 있습니다.

브라우저 지원
