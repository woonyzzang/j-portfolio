콘텐츠 분류 타입

  • 플로우 컨텐츠, 구문 컨텐츠에 속합니다.

참조

  • sup 요소의 첨자 텍스트는 기준선 위의 절반정도 위치에 문자로 브라우저에 렌더링 됩니다.
