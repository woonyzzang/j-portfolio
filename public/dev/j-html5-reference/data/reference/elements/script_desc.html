[요소가 속하는 범주]
  • 메타데이터
  • 플로우 컨텐츠
  • 구문 컨텐츠
[이 요소가 사용될 수 있는 문맥]
메타데이터를 쓸 수 있는 곳
구문 컨텐츠를 쓸 수 있는 곳
[이 요소가 포함할 수 있는 것]
src 속성이 없다면 type 속성의 값에 따라 다르며 스크립트 내용 제한을 만족해야 합니다.
src 속성이 있다면 요소는 반드시 비어 있거나, 비어 있지 않다면 스크립트를 문서화한 내용만을 포함할 수 있습니다. 스크립트를 문서화한 내용은 스크립트 내용 제한을 만족해야 합니다.
[요소에 사용할 수 있는 속성]
  • Global attributes
  • src
  • async
  • defer
  • type
  • charset
[DOM 인터페이스]
interface HTMLScriptElement : HTMLElement {
           attribute DOMString src;
           attribute boolean async;
           attribute boolean defer;
           attribute DOMString type;
           attribute DOMString charset;
           attribute DOMString text;
};

콘텐츠 분류 타입

  • 메타데이터, 플로우 컨텐츠, 구문 컨텐츠에 속합니다.
  • 메타데이터, 구문 컨텐츠가 사용될 수 있는 곳에서 사용될 수 있습니다.
  • 인라인으로 포함하거나 src 속성으로 외부 파일을 가져올 수 있습니다.
  • 스크립트 언어가 자바스크립트javascript가 아니라면 type 속성을 반드시 사용해야합니다.

참조

  • src 속성이 존재하는 경우, script 요소는 비어 있어야 합니다.
  • HTML 4.01 속성: "xml:space", HTML5에서 지원되지 않습니다.
